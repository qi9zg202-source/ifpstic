# “总执行优化策略”跳转 PRD

## 目标
在 `fab_cooling_cost_analysis.html` 页面中，点击“总执行优化策略 12,900条”的超链接，跳转到 `fab_ai_strategy_package_detail.html` 并携带筛选参数，实现详情页初始化。

## 跳转链接
- 目标地址：`fab_ai_strategy_package_detail.html?unit=month&start=2026-01&end=2026-02`

## 携带参数
- `unit`: 统计时间粒度
  - 取值：`month`
- `start`: 起始月份
  - 取值：`2026-01`
- `end`: 结束月份
  - 取值：`2026-02`

## 详情页初始化内容（fab_ai_strategy_package_detail.html）
### 1) 时间筛选组件
- 读取 URL 参数：`unit / start / end`
- 当 `unit=month` 时：
  - 时间控件显示为“月份范围”（或对应的时间输入控件显示 2026-01 ～ 2026-02）
  - 若页面使用 `datetime-local`：
    - 起始时间默认填 `2026-01-01 00:00`
    - 结束时间默认填 `2026-02-28 23:59`（2月按当年天数计算）

### 2) 筛选条件
- “下发状态”：默认选中 **已下发**
- “正负”：保持默认（ALL）

### 3) 数据展示
- 仅展示 **高效制冷机房** 下 **推荐给客户且客户已下发** 的策略包
- 时间范围限制为 `2026-01` 至 `2026-02`

### 4) 高亮/提示（可选）
- 在列表上方显示筛选说明：
  - `时间范围：2026-01 ～ 2026-02 | 下发状态：已下发`

## 失败与边界
- 若 URL 参数缺失：
  - 采用页面默认值
- 若时间参数非法：
  - 回退到页面默认时间
- 若无数据：
  - 表格显示“暂无数据”占位

## 验收标准
1. 在 `fab_cooling_cost_analysis.html` 点击“总执行优化策略 12,900条”可打开目标 URL。
2. 进入详情页后，时间范围与下发状态按参数初始化。
3. 列表仅展示已下发策略包，范围为 2026-01 至 2026-02。
