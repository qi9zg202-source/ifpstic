# 设备功率分析表增强功能说明

## 功能概述
为设备功率分析表的弹出窗口添加了设备类型多选功能，支持用户选择多种设备类型进行对比分析。

## 新增功能

### 1. 设备类型多选按钮
在弹出窗口的查询条件区域添加了设备类型选择区域，包含四种设备类型：
- 🔧 **冰机** (蓝色主题)
- 🔧 **冷冻泵** (绿色主题) 
- 🔧 **冷却泵** (橙色主题)
- 🔧 **冷却塔** (紫色主题)

### 2. 智能自动选择
- 点击表格中的具体设备行时，会自动选中对应的设备类型
- 默认只显示当前点击行对应的设备类型数据
- 清除其他设备类型的选择状态

### 3. 多设备对比显示
- 支持同时选择多种设备类型进行对比分析
- 每种设备类型使用不同的颜色和线型：
  - **实线**：功率数据
  - **虚线**：占比数据  
  - **点线**：性能系数数据
- 图表标题动态显示已选择的设备类型

### 4. 增强的数据可视化
- **多Y轴显示**：
  - Y1轴：功率 (kW) - 蓝色
  - Y2轴：占比 (%) - 绿色
  - Y3轴：性能系数 - 橙色
- **颜色编码**：每种设备类型使用专属颜色系列
- **图例优化**：显示所有设备类型和指标的组合

### 5. 统计信息更新
- 统计面板显示所有选中设备的综合数据
- 包括平均功率、最大功率、平均占比、平均性能系数等
- 支持多设备数据的累加和平均计算

### 6. 导出功能增强
- CSV导出支持多设备类型数据
- 文件名包含所有选中的设备类型
- 数据列按设备类型分组显示

## 使用方法

### 基本操作
1. 点击设备功率分析表中的任意设备行
2. 弹出窗口会自动选中对应的设备类型
3. 可以手动勾选其他设备类型进行对比
4. 图表会实时更新显示选中设备的数据

### 多设备对比
1. 在设备类型选择区域勾选多个设备类型
2. 图表会显示所有选中设备的数据曲线
3. 不同设备使用不同颜色和线型区分
4. 统计信息显示综合数据

### 数据导出
1. 选择需要导出的设备类型
2. 设置时间范围和数据粒度
3. 点击"📊 导出"按钮
4. 下载包含所有选中设备数据的CSV文件

## 技术实现

### 前端交互
- 使用复选框实现设备类型多选
- JavaScript动态更新选择状态和显示文本
- 实时响应用户选择变化

### 数据处理
- 为每种设备类型生成独立的数据序列
- 支持多设备数据的合并和统计计算
- 动态创建Highcharts图表系列

### 图表渲染
- 使用Highcharts多Y轴配置
- 动态生成不同颜色和样式的数据系列
- 支持缩放、导出等交互功能

## 用户体验优化
- 直观的颜色编码和图标设计
- 响应式布局适配不同屏幕尺寸
- 实时反馈和状态提示
- 一键操作和智能默认选择

这个增强功能大大提升了设备功率分析的灵活性和实用性，用户可以方便地进行单设备深度分析或多设备对比分析。
